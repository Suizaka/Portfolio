🚀 Projects

A showcase of my creations, explorations, and innovations.

UltraSonic Detector

🔍 Real-time Ultrasonic Radar System using Arduino and Processing for dynamic object detection and live visualization.

Fire Safe Now

A real-time fire threat detection and response website with live maps and interactive UI, built in 48 hours for NASA’s Space Apps Hackathon 🚀🌍

Obstacle Alert System

An Arduino-powered project that uses an ultrasonic sensor 🚨 to detect nearby objects 🧱 and triggers a buzzer 🔔 and LED 💡 when the object comes too close, ensuring quick alerts for safety.

🔧📉 OPAMP 741 Analog Kit

A dual-function hardware project showcasing integrator & differentiator circuits, fully hand-soldered on zero PCB 🛠️, enclosed in a wooden box 🪵, and equipped with custom-made BNC 🔗 and banana 🔌 connectors for lab testing.

🔒 ATmega32 Passcode System

A secure access system using a 4x4 keypad, LCD, buzzer, and LEDs 🔢💡. Built in C, it validates password entries and gives real-time feedback with alerts and display messages.

🏡🔋 Smart Home Automation (ESP32)

A solar-powered smart system with Bluetooth app control 📱, Google Home voice integration 🎙️, manual buttons, motion & sound sensors 🕵️‍♂️👏, and flame alert safety 🔥 — all simulated on a cardboard home model for real-world prototyping.
