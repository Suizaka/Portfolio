Built as part of the Microprocessor & Microcontroller subject in 4th semester, this project is a basic yet functional digital security system. It uses a 4x4 keypad to input a password, displays user interaction messages on an LCD screen, and controls access via LED indicators and a buzzer.

The system mimics a real-world access control module, offering a compact, user-friendly design and logical flow for password entry and validation.
